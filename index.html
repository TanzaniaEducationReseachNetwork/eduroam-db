eduroam-db by stas

A possible implementation of Eduroam Monitoring Database

Please consult the official webpage regarding specifications and other documentation.

Dependencies

PHP5 SQLite3 PHP5 SQLite PDO module

Install

Download the latest tar|zip package from http://github.com/stas/eduroam-db Extract the contents in a directory accessible from a web browser. Got to app/config.php and edit the basic settings for web address and database location. According to the path you set in the config.php populate the database like this: $ sqlite3 path/to/database.db < db/schema.sql After it got to the web location of the installation and register.
